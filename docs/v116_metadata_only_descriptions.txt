V116: Adds a metadata-only description layer. Existing vanilla Symbol/Item/Essence database entries keep their real LBAL names/descriptions/icons, but entries that are not AP-allowed are flagged ap_metadata_only_v116 and skipped by database backfills. This lets locked content keep default descriptions without putting it into Add Symbol/Add Item/Add Essence pools. Patch marker is AP_LIVE_POOL_PATCH_V116.
