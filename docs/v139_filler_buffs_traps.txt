V139: Adds optional filler-replacement buffs/traps with individual toggles. Buffs/traps replace configured percentages of filler checks only. Added buffs: start with a destroy token, start with $5, start with a reroll token, and choice of 20 symbols. Added traps: half money, force payment, and add a Dud symbol. The client writes received AP effect items to ap_state.json, and Main.tscn applies each received effect once using ap_effects_applied.json. Patch marker is AP_LIVE_POOL_PATCH_V139.
